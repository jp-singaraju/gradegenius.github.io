Privacy Policy

  1. Receive data from the FISD HAC Gradebook system using login credentials
  2. Store the login credentials and extra user details in the local phone files

This app does not contain any ads. We do not disclose any information to third parties. Also, we are not storing any user credentials, data, or details. The only storage of information is only on the local user device. The storage of data and extra collection information is listed below. This app complies with all policies regarding children.

Information and Data Collection

This app is intended for FISD students who wish to view their grades by phone. This may include elementary school, middle school, and high school students. In order to completely access the features within this app, the user must provide the login credentials to his/her FISD HAC Gradebook. The app then contacts the HAC server and processes the information in an encrypted server. The data is then aesthetically displayed on the user's phone. The information stored on the local phones files is not visible by anyone except the user. We cannot access your information in any way possible. However, the app does store information on the local device. All content within the app is appropriate for use by children and does not contain any inappropriate information. All requirements satisfy the Google Play Console Families Policy for children: Google Play Console Family Policy

Cookies

Cookies are small files that store data within them to make requests to a server or for data to be processed on a website. This app does use cookies and the use of cookies is completely secure as they are encrypted.

Sessions

A session is used to persist user data so that the information can be displayed on the user phone without logging out. This app does use sessions to persist user data until the user becomes inactive. The sessions contains cookies that make requests to the HAC gradebook.

Storage of Local Information

Information that is stored locally is only accessible by the user. This information stored on the local device includes user login credentials, user preferences within the app, and user HAC information. All the data stored locally is only used to provide certain app features.

Receive Complete Network Access

In order for this app to run, it must have internet/network access. The user may turn of internet on the device at any time, but this app will not perform as the features will not work.
